Query Execution

Key combination What it does
⌘ E Execute query
⌥ ⌘ E Cancel current query execution
⌘ U Change current database
⇧ ⌘ C Change database server for the current Queries window
⌘ R Reload database objects sidebar
⌘ I Explain currently selected database object in objects sidebar.
Generates create table script for tables, alter script for views, stored procedures and functions.
⌥ ⌘ F Search database objects in current database.
Uses regular expression to matches full object name (schema.name) for all objects in current database.

Go To

Key combination What it does
⌥ ⌘ ↑ Go to query text view. Returns focus to query text view.
⌥ ⌘ ↓ Go to next results view tab. Cycles through results view tabs (table results, text results, messages) when pressed repeatedly
F9 Go to database objects sidebar
F10 Go to Table results
F11 Go to Text results
F12 Go to Messages
⌃ ⌘ ↑ Cycle through: maximized query view, maximized results, query and result split equally
⌃ ⌘ ← Show/hide database objects sidebar
⌘ 1
⌘ 2
...
⌘ 0 		Go to first query tab
Go to second query tab
...
Go to tenth query tab
⌃ ⇥ Cycles through all active views (query text view, query results, objects sidebar)
⌃ ⇧ ⇥ Same as previous but cycles backwards.
